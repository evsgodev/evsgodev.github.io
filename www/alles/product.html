Модульная программа мебели для кухни «Марсель».

Классическая кухня, выдержанная в стиле «кантри». Фасады изготовлены из массива дуба со старением. каркасы — из ДСП. Краситель при изготовлении кухни не используется. Дуб имеет натуральный оттенок. После нанесения лака наносится патина на контур фасада. Патина закрепляется лаком. Впечатление старины подчеркивают грубые ручки «под ковку» и накладные штыревые петли, выполненные под «состаренную бронзу». Кроме того, кухню отличают рамочный каркас, разнообразные элементы с арочными формами и ящиками, декоративные двери, предназначенные для установки на боковых стенках, оригинальный держатель для полотенец, резная полка для вытяжки, открытая сушка для посуды, декоративные элементы с профильной обработкой для столешниц. Ящики и метабоксы комплектуются фурнитурой «Люкс» с доводчиками. Фабрика предлагает дополнительно изготовление декоров из массива дуба для столешниц из керамической плитки.

Кухню «Марсель» дополняет большая коллекция, состоящая из обеденных столов, буфетов, тумб различных размеров, которые в отличие от кухни полностью изготовлены из массива, а также мягкая мебель.
